## ⚠️ 真相以数据库为准（每次先实查，别信手维护的"已做"列表）

手维护的"已做"清单会过时。判断一个世界**是否真做成**，跑这条（Docker 项目名 `talealive`，DB=`inkwild`）：

```bash
docker exec talealive-db-1 psql -U postgres -d inkwild -t -A -F" | " -c "
select w.name, w.status,
  (select count(*) from scripts s where s.world_id=w.id) as scripts,
  (select count(*) from world_characters c where c.world_id=w.id) as chars,
  (select count(*) from game_sessions g where g.world_id=w.id) as sessions
from worlds w order by w.created_at;"
```

判定口径：**剧本数 ≥ 3 且有真实对局**才算"真做成、计入排除"；`scripts=1` 的多是早期测试生成的空壳，**不算已做，可重做**。
