## A1《紫禁初风云》（大篇章本 · 华妃争锋弧）

**推荐视角**：甄嬛（主）/ 华妃 / 安陵容 / 沈眉庄（alt-POV 各出一版）
**忠实事件链（events 骨架，按时间线触发）**：
1. 选秀入宫，初居碎玉轩（开场）
2. 倚梅园除夕"偶遇清河王"（实为皇帝）——初得圣意
3. 惊鸿舞得盛宠（似纯元，皇帝动心）
4. 华妃立威：一丈红杖责夏冬春（华党跋扈）
5. 华党构陷：木薯粉/温宜中毒嫁祸甄党 → 反击
6. 沈眉庄被诬"假孕"（皇后华妃合谋）
7. 年羹尧获罪、华妃失势；欢宜香=麝香致不孕真相浮出
8. 皇后设局"纯元旧衣"→ 甄失宠（弧线高潮转折）

**双结局**：
- 复刻原著：华党倒台、甄因纯元旧衣心灰，决意出家甘露寺。
- 改写命运（分歧）：① 识破纯元旧衣陷阱、未失宠，留宫续斗皇后；② 提前坐实欢宜香、更早扳倒华妃、保住圣宠；③ 保下沈眉庄免遭诬陷；④ 翻车：树敌太多被联合构陷打入冷宫。

**信息隔离锚点（NPC 秘密，驱动行为非给玩家解谜）**：华妃不知欢宜香=麝香；皇后藏害纯元旧账；皇帝视甄为纯元替身。

## S2《滴血验亲》（名场面小本 · 自包含 · demo 首选）

**背景**：回宫后，祺贵人联合皇后当殿告发甄嬛与温实初私通、六阿哥非龙裔，要求滴血验亲。
**推荐视角**：甄嬛（主，被告、信息受困）/ 温实初（太医，能验能牺牲）/ 旁观低位嫔（信息少但能游走查水）
**忠实事件链**：
1. 祺贵人当殿发难，告发私通（开场）
2. 第一次滴血验亲：血相融（有人在水中做了手脚）→ 危局
3. 玩家调查/反击：查验那盆水、找证人（温实初、宫人、采蜜小太监）
4. 揭穿水被动了手脚（加白矾/盐）→ 反转
5. 温实初自证（极端：自宫明志）或找出加料之人
6. 反咬：揪出祺贵人背后的皇后

**双结局**：
- 复刻原著：拆穿做手脚、祺贵人倒台、温实初自宫，甄脱险并反将皇后一军。
- 改写命运（分歧）：① 提前查出水中猫腻、不靠温实初牺牲即脱险；② 保住温实初不自宫；③ 未能拆穿 → 被坐实私通赐死/打入冷宫（失败结局）；④ 顺势翻出皇后害纯元旧案、当场扳倒皇后（高光分歧）。

**信息隔离锚点**：谁在水里动手脚（皇后党指使）；温实初对甄之情。
