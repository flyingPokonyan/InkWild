## API Design

### Key Functions

**`is_transient(exc: BaseException) -> bool`**
- Returns `True` for `TransientError` instances
- Returns `True` for exception class names in `TRANSIENT_EXCEPTION_NAMES` (APITimeoutError, RateLimitError, etc.)
- Returns `True` for exceptions with `status_code` or `status` attribute in range [500, 600)
- Returns `False` for all other exceptions (4xx, JSON errors, ValueError, etc.)

**`async with_transient_retry(...) -> T`**
- Parameters:
  - `coro_factory: Callable[[], Awaitable[T]]` — factory to call fresh on each attempt (not single coroutine)
  - `max_attempts: int = 3` — total attempts including first attempt
  - `backoffs: tuple[float, ...] = (1.0, 3.0)` — sleep durations between retries (uses last value if exhausted)
  - `on_retry: Callable[[int, int, BaseException], Awaitable[None]] | None = None` — optional callback fired before each retry
- Behavior:
  - Retries only for transient exceptions (detected via `is_transient()`)
  - Non-transient exceptions raised immediately (no retry)
  - Original exception re-raised after max_attempts exhausted
  - Callback failures logged but do not interrupt retry logic

## Implementation Notes

### Type Safety
- Full type hints with `TypeVar[T]` for generic return type
- Proper `Awaitable` typing for coroutine factories
- Modern union syntax (`BaseException | None`) per CLAUDE.md standards

### Error Handling
- `ValueError` if `max_attempts < 1` (catch configuration errors early)
- Callback failures logged via structlog, do not interrupt retry loop (robustness)
- Last exception always raised verbatim (no wrapping, spec §6.1 requirement)

### Async Patterns
- Factory is `Callable[[], Awaitable[T]]` not a coroutine (allows fresh invocation each retry)
- Uses `asyncio.sleep()` for backoff (non-blocking)
- Properly awaits on_retry callback

### Module Independence
- No dependencies on `v2_agent` or other services
- Ready for direct integration in any LLM call site
- Can be imported as `from services.world_creator_retry import with_transient_retry, TransientError`
