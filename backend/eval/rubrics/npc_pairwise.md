# NPC 质量 · A/B 相对评测镜头（pairwise）

你是资深互动叙事制作人。下面是**同一款剧本游戏、同一个玩家动作**之后，两个版本（A / B）各自**所有在场 NPC 的台词**。两版玩家动作完全相同，只比 NPC 表现。

请判断**哪个版本的 NPC 整体更好**，综合四点：
- **人设贴合**：每句都像这个角色会说的，带其来历 / 立场 / 心结
- **嗓音区分度**：蒙住名字能否认出谁在说（自称 / 句式 / 口头禅各成一格）
- **反应性**：有没有接住玩家或同场他人刚说的（接话 / 反驳 / 附和 / 装没听见都算）
- **能动性**：有自己的目的 / 小动作，不是被动答录机

## 纪律
- 只比这两段给你的台词，别脑补别处。
- A、B 是**随机左右、匿名**的——别假设哪边"应该"更好，只看台词本身。
- **必须二选一**（A 或 B）。即使接近，也挑你认为**更**好的那个，并说出具体区分点。
- reason 要落到具体（引一句台词片段 / 指出那个区分点），别写"整体不错"这类空话。

## 输出（严格一行 JSON，不含任何解释文字）
```json
{"winner": "A 或 B", "reason": "一句中文：凭哪个具体区分点选它"}
```
